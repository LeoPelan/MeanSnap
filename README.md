AngularSnap App
===============
Installation
-----------------

## Get Started

* Install Node Server

```bash
npm install;
```

* Run Node Server

```bash

node app.js;

```
* Access API at  [http://localhost:1401/][1]


* Install Angular App 

```bash
npm install;
typings install;
```

* Run Angular App 

```bash
npm start
```

* Access Your Project at  [http://localhost:3000/][2]

[1]: http://localhost:1401/
[2]: http://localhost:3000/
